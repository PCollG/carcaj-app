# Carcaj

Monorepo de la app de puntuación de tiro con arco.

- `apps/mobile` — app Expo/React Native ([detalles](apps/mobile/README.md))
- `apps/api` — backend Node + Express + Prisma ([detalles](apps/api/CLAUDE.md))

Ver `CLAUDE.md` para el dominio, el modelo de datos y la arquitectura del proyecto.

## Empezar

```bash
npm install       # instala dependencias de ambos workspaces
npm run mobile    # arranca Expo
npm run api       # arranca el backend
```
